# Pacfoster

A cleaning script similar to 'debfoster', using the pacman package manager.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "pacfoster"
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install pacfoster

## Usage

```
$ pacfoster
```
